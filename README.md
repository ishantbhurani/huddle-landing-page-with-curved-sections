## Run Locally

Clone the project

```bash
  git clone git@github.com:ishantbhurani/huddle-landing-page-with-curved-sections.git
```

Go to the project directory

```bash
  cd huddle-landing-page-with-curved-sections
```

Run `index.html`

```bash
  <browsername> index.html
```

E.g.

```bash
  firefox index.html
```

```bash
  google-chrome index.html
```
